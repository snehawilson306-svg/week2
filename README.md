# Configuration-Driven Website System

## Project Overview

This project implements a configuration-driven website using HTML, CSS, and JavaScript.

The website dynamically reads settings from JSON configuration files and updates:
- Layout
- Navigation
- Colors
- Footer

## How to Run

Use Live Server in VS Code.

Open:
frontend/index.html

To test different configurations:

index.html?config=site-config-blog.json  
index.html?config=site-config-portfolio.json  
index.html?config=site-config-business.json  

## What I Learned

- Using fetch() to load JSON
- DOM manipulation
- Dynamic UI generation
- Separation of structure and configuration
